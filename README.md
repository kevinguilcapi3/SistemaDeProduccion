# Sistema de Producción en C

## Descripción

Este proyecto consiste en un programa modular desarrollado en lenguaje C para gestionar productos dentro de una fábrica.

El sistema permite:

- Ingresar productos
- Editar productos
- Eliminar productos
- Calcular tiempo y recursos de producción

El programa fue desarrollado aplicando programación modular mediante:

- Archivo principal (`main.c`)
- Archivo de funciones (`SistemaProduccionFunciones.c`)
- Archivo de cabecera (`SistemaProduccion.h`)

---

## Tecnologías utilizadas

- Lenguaje C
- Programación modular
- Funciones
- Arreglos
- Estructuras de control

---

## Archivos del proyecto

| Archivo | Función |
|---|---|
| main.c | Controla el menú principal |
| SistemaProduccion.h | Declaración de funciones |
| SistemaProduccionFunciones.c | Desarrollo de funciones |

---

## Compilación del programa

```bash
gcc main.c SistemaProduccionFunciones.c -o programa
```

---

## Ejecución

```bash
./programa
```
